# [rubrica-presenze.user.js](https://github.com/acavalin/tp_unipd/raw/refs/heads/main/rubrica-presenze.user.js)

Nella [rubrica di ateneo](https://www.unipd.it/contatti/rubrica) aggiunge accanto
al nome delle persone un link all'[app delle presenze del personale](https://apex.cca.unipd.it/pls/apex/f?p=1300:120).

Nell'[app delle presenze del personale](https://apex.cca.unipd.it/pls/apex/f?p=1300:120)
aggiusta automaticamente i filtri di selezione.
